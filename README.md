<h1>Number Analyser</h1>

In this application you provide as many number as you wants, and then it gives you some information about it, like: 
 - the quantity of numbers given
 - the biggest number
 - the smaller number
 - the sum of them all
 - and the average

It was used for this application:
 - HTML
 - CSS
 - JvaScript
